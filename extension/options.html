The local server aggregates push data from this extension with server-side collectors (Bitwarden-backed ZAI key, Codex auth.json). It's optional — the extension popup works standalone.
cd usage-monitor && node server.mjs
Run from the nana/usage-monitor directory. Keep it running in a terminal tab.

For agents / scripts

When the server is running, agents can read the merged view over HTTP. JSON only, no auth.
curl -s http://localhost:3030/api/usage
curl -sX POST http://localhost:3030/api/refresh
Server re-runs ZAI + Codex collectors on every call. Ollama/Claude use the last push from this extension — open any dashboard page (or trigger via POST /ingest/:provider) to update them.
